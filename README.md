# Прототип автоматизации ПВЗ

## Назначение проекта

Проект демонстрирует работу информационной системы пункта выдачи заказов. Приложение позволяет вести справочник товаров, принимать товары на склад, создавать и обрабатывать заказы, выдавать заказы получателям, просматривать складские операции и формировать отчеты.

## Стек

- ASP.NET Core MVC
- Entity Framework Core
- PostgreSQL

## Требования для запуска

- .NET 8 SDK
- PostgreSQL

## Настройка БД

Для запуска нужна база данных PostgreSQL:

```sql
CREATE DATABASE pvz_prototype;
```

Строка подключения находится в файле `Prototype/appsettings.json`:

```json
"DefaultConnection": "Host=localhost;Port=5432;Database=pvz_prototype;Username=postgres;Password=postgres"
```

При необходимости измените пользователя, пароль, хост или порт под локальную установку PostgreSQL. При первом запуске приложение применяет миграции базы данных и добавляет стартовые учетные записи и демонстрационные данные.

## Команды запуска

```bash
dotnet restore Prototype/Prototype.csproj
dotnet run --project Prototype/Prototype.csproj
```

После запуска приложение будет доступно по адресу, который выведет `dotnet run` в консоли.

## Стартовые учетные записи

| Логин | Пароль | Роль |
| --- | --- | --- |
| `employee` | `12345` | Сотрудник |
| `manager` | `12345` | Управляющий |
| `lead` | `12345` | Руководитель |
| `sysadmin` | `12345` | Системный администратор |

## Основной демонстрационный сценарий

1. Войти в систему под пользователем `manager` или `employee`.
2. Открыть раздел «Склад» и выполнить приемку товара.
3. Открыть раздел «Заказы» и создать заказ на доступное количество товара.
4. Перевести заказ в сборку.
5. Подтвердить комплектацию заказа.
6. Выдать заказ и указать документ получателя.
7. Открыть раздел «Отчеты» и проверить выданный заказ и журнал складских операций.

Для демонстрации администрирования можно войти под `manager` или `sysadmin`, открыть раздел «Администрирование», создать нового пользователя или товар, а затем удалить запись, если она не используется в заказах и складских операциях.
